PROJECTS

AMATEUR RADIO PRACTICE EXAMS

I created the site amateur-radio.school to help people study for the amateur radio licensing exams. As I mentioned in the about me section above, I uploaded all the questions from the question pools to Contentful. I then created a front end for the site using React. For the styling I used Styled Components. The site is hosted on Netlify.

The user can choose between three different license levels for the exam they want to practice. The exams are then generated and the user can work their way through the test. The user is given a score at the end and notified if they pass or fail. The grading and the questions chosen are based on the requirements from the FCC. If you are interested you can read more at arrl.org.

Like most of my projects, this is still a work in progress. There are a couple of things I want to change. The most difficult part was converting the questions to JSON and getting them into a CMS, and now that that is done creating any kind of front end is possible.

I am planning on redoing the site using Next.Js rather than using Create React App. Using Next.Js will allow me to more easily generate the exams, right now they are generated each time a user starts an exam which can be slow at times.

Some Other Features I Want To Add:

  • Ability for users to save progress in exams.
  • Marking questions wrong or right as the user takes the exam.
  • A summary of the wrong answers and links to help the user study.
  • User accounts.

You can view the working site here. The code is available on Github.

CANCEL REMINDER

Cancel Reminder is a website that lets users keep track of any subscriptions they have. A user can add their subscriptions to services such as Netflix, Amazon, etc and Cancel Reminder will tell them how many days they have until they will be charged again for the service.

Cancel Reminder was created with Next.Js and Firebase. I used Firestore for storing the user's data and Firebase's authentication for user management. Vercel is used to host the site since they make hosting Next.Js so easy.

I first started the site by creating a design in Figma before moving on to the actual coding. I created all custom components: collapsible containers that work on mobile and desktop, a working calendar, mobile collapsing menu, and others. The site is also fully responsive and uses CSS modules for the styling.

This site is still a work in progress but the flow of the site is working pretty good. I have a welcome page that directs a user to sign up for an account. There is a sign up page and also a sign in page for existing users. Once signing in you are redirected to the dashboard that displays all your subscriptions.

There are some limitations to what this app can do though. It cannot actually access a user's Netflix or Amazon account to cancel for them, so it is up to the user to manually cancel and then mark the item as cancelled. But I still think the app could be useful for keeping track of subscriptions.

There is a working demo of the site at cancel-reminder.vercel.app. The code is available on Github.
