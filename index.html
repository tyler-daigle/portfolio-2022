PROJECTS

AMATEUR RADIO PRACTICE EXAMS

I created the site amateur-radio.school to help people study for the amateur radio licensing exams. As I mentioned in the about me section above, I uploaded all the questions from the question pools to Contentful. I then created a front end for the site using React. For the styling I used Styled Components. The site is hosted on Netlify.

The user can choose between three different license levels for the exam they want to practice. The exams are then generated and the user can work their way through the test. The user is given a score at the end and notified if they pass or fail. The grading and the questions chosen are based on the requirements from the FCC. If you are interested you can read more at arrl.org.

Like most of my projects, this is still a work in progress. There are a couple of things I want to change. The most difficult part was converting the questions to JSON and getting them into a CMS, and now that that is done creating any kind of front end is possible.

I am planning on redoing the site using Next.JS rather than using Create React App. Using Next.Js will allow me to more easily generate the exams, right now they are generated each time a user starts an exam which can be slow at times.

Some Other Features I Want To Add:

  • Ability for users to save progress in exams.
  • Marking questions wrong or right as the user takes the exam.
  • A summary of the wrong answers and links to help the user study.
  • User accounts.
